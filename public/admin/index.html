📋 Hướng dẫn Quản lý Ngân hàng đề

  1. Nhấp vào một trong các khung kỹ năng ở trên để xem chi tiết câu hỏi và vào giao diện nhập liệu.
  2. Chọn câu hỏi bất kỳ trong danh sách bên trái để Chỉnh sửa nội dung câu hỏi đó.
  3. Nhập thủ công câu hỏi ở khung bên phải và bấm Lưu câu hỏi để cập nhật hoặc thêm mới.
  4. Bạn cũng có thể tải file Excel mẫu bằng nút Tải Template và tải lên bằng Import Excel để thêm hàng loạt.
